# ShowBlocks

ShowBlocks is an educational game meant to make learning SQL more active and intuitive! It provides an SQL sandbox that allows users to actively see the effects of queries on their databases.


Pan and zoom to your heart's content!

![Movement and Cam](/img/exp1.gif)

Write queries according to the given context, all with interactive UI!

![Interaction](/img/exp2.gif)

Robustly handle errors and learn from your mistakes!

![Error handle](/img/exp3.gif)

Write increasingly complex queries, using commands such as SELECT:

![select](/img/exp4.gif)

DELETE:

![delete](/img/exp5.gif)

And INSERT INTO:

![insert](/img/exp6.gif)

The currently available version of ShowBlocks is a proof of concept, as there are a lot of areas still being worked on, in order to make ShowBlocks the best possible game it can be!
